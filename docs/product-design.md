# Product Design Notes

## Core Screens (Phase 1)

1. Landing page
   - Game cards (snake, tetris, space invaders).
   - Quick play button.
   - Last played and personal best indicators.
2. Game details modal/page
   - Controls summary.
   - Difficulty and start options.
3. In-game shell overlay
   - Pause/resume/restart.
   - Return to launcher.
4. Profile/settings
   - Local player name.
   - Audio/preferences.

## UX Consistency Rules

- Keep controls and start flow predictable across games.
- Use consistent key mappings where genre allows.
- Show essential status info only (score, lives, level/timer).

## Accessibility Baseline

- Keyboard-first navigation in launcher UI.
- High contrast option.
- Reduced motion option for shell transitions.
- Clear focus states for all actionable UI.

## Visual Direction

- Arcade-inspired theme with modern readability.
- Distinct branding per game card while keeping shared shell identity.
- Responsive layout for desktop and mobile landscape.
